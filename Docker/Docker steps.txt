1. Install docket by following the steps on the official docker website
2. Install docker compose by following the steps on the official website by downloading it from github
3. Grant full right to the current user in docker
	sudo usermod -aG docker ubuntu
4. Make docker automatically start when the system starts
	sudo systemctl enable docker
5. Copy the project files to the server (/home/ubuntu/elenora-test)
6. Create the project image (cd to the elenora-test folder)
	docker build --tag elenora-test
	
	
Useful commands
	pwd  
	docker image ls 
	docker container ls
	docker-compose up -d
	docker-compose down --rmi all
	docker build --tag elenora-test .
	docker image prune --all //remove all unused images
